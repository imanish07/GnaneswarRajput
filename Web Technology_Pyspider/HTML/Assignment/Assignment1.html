Web Development

Web technology is used to designing and developing the frontend of webpage. It is completly developers responsibility to keep webpage attractive.

HTML

HTML is used for building the structure of the webpage. Without HTML we can't use css and javascrit.

CSS

CSS is used to improve the style of the webpage, without CSS we cannot improve HTML elements.

Javascript

Javascript is used to improve the logic of the webpage. Without javascrit we cannot give logic to our webpage.
